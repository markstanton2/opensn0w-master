## 1. DESCRIPTION

opensn0w is an open-source jailbreaking utility designed for 
platform-independence.

## 3. HOW TO INSTALL

- Run `./autogen.sh`
- Run `./configure --prefix=/your/install/dir/here`
- Run `make; make install`
- Done.
